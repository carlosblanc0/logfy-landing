## Run Locally

Clone the project

```bash
  git clone https://github.com/carloswhite/logfy-landing.git
```

Go to the project directory

```bash
  cd logfy-landing
```

Install dependencies

```bash
npm install -g live-server
```

Start the server

```bash
  npm run start
```
